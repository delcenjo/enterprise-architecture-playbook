# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-22

### Added
- React 19 + Vite + TypeScript enterprise decision UI (frontend)
- PDF/Excel dossier builder with premium HTML templates (output_generator)
- Comprehensive README with architecture diagrams, setup guide, and usage documentation
- Full test coverage with 37 verification suites across all engine pillars

## [0.8.0] - 2026-02-19

### Added
- Developer onboarding knowledge base with decision trees and case studies
- Financial operations pillars: Supply Chain Security (SBOM/SLSA), LTV Dynamics, Unit Economics
- Product management pillars: TPM/RICE prioritization, OKRs/DORA metrics, Tradeoff Analysis
- Organizational pillars: Team Topologies, Platform Engineering, Cultural Fit Assessment, Senior Evaluation
- Engineering practices pillars: Tech Debt Management, Code Review Automation, Scalability Patterns, Architecture Fitness Functions
- Performance optimization pillars: Load Testing (k6/Gatling), Profiling, DB Optimization, Frontend Web Vitals
- SRE & Observability pillars: SRE/SLO Management, Distributed Tracing (OpenTelemetry), Chaos Engineering
- Deployment & Infrastructure pillars: Blue-Green/Canary Strategies, GitOps (ArgoCD/Flux), Infrastructure as Code (Terraform/Pulumi)

## [0.5.0] - 2026-02-06

### Added
- Architecture patterns knowledge base with ontology, decision trees, case studies, and tradeoff matrices
- Regulatory compliance knowledge base covering GDPR, DORA, PSD2/SCA, AML/KYC, CNMV/MiCA, and BdE 4/2017
- Base operational ontology and decision framework for 35+ engineering pillars

## [0.3.0] - 2026-02-01

### Added
- 6-layer deterministic processing pipeline (Input → Core Engine → Calculation → Visual → Validation → Narrative)
- Core orchestrator with FastAPI, Pydantic state management, and workflow manager
- Static assets including architecture diagrams, financial projections, risk heatmaps, and report templates
- LLM reasoning engine with multi-provider support (Claude, GPT, Gemini, Ollama fallback)
- Simulation engine for scenario modeling and cost projections
- Workflow engine for structured analysis pipelines

## [0.1.0] - 2026-01-17

### Added
- Cloud pricing query API service (FastAPI + PostgreSQL)
- AWS price data ingestor with validation pipeline and normalization
- PostgreSQL database schema with pricing tables and indexes
- Docker Compose multi-service orchestration (6 services)
- Initial project structure with comprehensive .gitignore
